# StatSigCalculator — A/B Test Sample Size Calculator

Statistical significance calculator for A/B testing. Determine the required sample size for your experiments with confidence.

## Features

- Calculate minimum sample size for A/B tests
- Support for various statistical parameters (alpha, power, effect size)
- Interactive UI with real-time results

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
- PostCSS + Autoprefixer

## Quick Start

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
